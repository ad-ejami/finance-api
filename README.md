# finance-api
API regarding finance program

Create your custom env

```bash
python3 -m venv env
```

Start the project env

```bash
source env/bin/activate
```

Stop enviroment project

```bash
deactivate
```

Store dependencies

```bash
pip3 freeze > requirements.txt
```

Install dependencies

```bash
pip3 install -r requirements.txt
```

run the app
```bash
fastapi dev
```
